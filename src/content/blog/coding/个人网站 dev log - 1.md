# 创立网站
创立网站模板的 命令行是 
```bash
npm create astro@latest -- --template minimal
```
创立之后整体看起来很简约，没有之前用 Vite 构建的网站那么花里胡哨。

网站下面有一个 astro 控制台，通过如下命令进行禁用。

```bash
npx astro preferences disable devToolbar
```
# 页面修改
## 通用
### 修改网页语言设置
模板创立之后，网页默认语言属性是英文，每次打开都会跳出要不要翻译页面的提示，比较烦人，修改：

```html
<!-- 备忘：英语为 "en"，繁体中文为"zh-TW" -->>
<html lang="zh-CN">
```
### 网页通用布局实现
个人目前打算把网页做成类似微信或者 QQ 聊天框的形式，目前暂定的布局是：
1. 上栏，作为控制栏和信息栏。
2. 下栏分为两栏。
   1. 左栏为索引，可以为多级。
   2. 右栏为 preview 的内容。
